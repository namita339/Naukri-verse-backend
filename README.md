# Naukri Verse Backend

Backend for a Job Portal application built using Node.js and Express.

## 🚀 Features
- User authentication (login & register)
- Job posting system
- Job application management
- Dashboard APIs
- RESTful backend structure

## 🛠 Tech Stack
- Node.js
- Express.js
- HTML
- CSS
- JavaScript

## 📁 Project Structure
models/
routes/
public/
server.js
package.json
## ▶️ Run the Project

Install dependencies:

npm install

Start the server:

node server.js
